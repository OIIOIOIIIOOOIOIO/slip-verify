Slip Verify ถูกพัฒนาขึ้นเพื่อเป็นเครื่องมือช่วยตรวจสอบสลิปโอนเงินให้เข้าถึงได้ง่าย โดยเปิดให้ใช้งานฟรีสำหรับทุกคน

## คำแนะนำในการใช้งาน

เพื่อความถูกต้องสูงสุด ควรตรวจสอบข้อมูลสำคัญร่วมกันทุกครั้ง เช่น

* ชื่อบัญชีผู้รับเงิน
* เลขบัญชีหรือพร้อมเพย์
* จำนวนเงิน
* วันและเวลาทำรายการ
* แหล่งที่มาของสลิป
* ความสอดคล้องของข้อมูลกับรายการจริง

การตรวจสอบผ่านระบบเป็นเพียงส่วนหนึ่งของกระบวนการยืนยันข้อมูล ผู้ใช้งานควรตรวจสอบกับเจ้าของบัญชีหรือผู้รับเงินร่วมด้วยเสมอ
